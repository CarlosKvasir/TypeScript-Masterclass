## Configurando ESlint

Adicione como dependecia de desenvolvimento o eslint e estes plugins:

``` sh
yarn add -D eslint @typescript-eslint/parser @typescript-eslint/eslint-plugin
```

Configure o ESLint 
```sh
yarn eslint --init
```
Enforce >> import/export >> para node (Standard) >> salvar em JS
